利用tushare获得想要的文章

描述：

Tushare是一个免费、开源的财经数据接口包。
利用其可得到股票分类数据、基本面数据、宏观经济数据以及电影票房等数据。
网址链接

安装：

Tushare的安装以Python/pandas/lxml为前提
可使用PIP方法
在MAC的环境下需要sudo给予权限
具体：（以MAC为例）

sudo pip install pandas ...

输入密码给予权限

sudo pip install lxml （步骤同上）
......

（步骤同上）
sudo pip install tushare
…（步骤同上）

安装完成

使用：

点开documentation,直接使用里面的代码以获得资料
以获取每日电影票房数据为例：

import tushare as ts
df = ts.day_cinema() #取上一日全国影院票房排行数据
#df = ts.day_cinema('2015-12-24') #取指定日期的数据
df.head(10)

则可得到等数据

后续

此时可接下来做描述性统计或者回归分析
再用seaborn画图
若想得到更多的数据
可将df.head(10)括号里的数字改为其他数据
